每次完成任务，都在创建任务日志。文件路径是 `docs/logs/YYYYMMDD-中文任务名.md`

## 关键目录结构

- `docs/` 项目文档
- `docs/logs/` 任务日志
- `scripts/` 脚本
- `src/` pocketstack 项目源代码
- `src/modules/` pocketstack 模块
- `src/modules/slides/` 幻灯片模块
- `src/modules/archigram-static/` 架构图模块

## skills

skills存放在 `.skills`目录下：

- `agent-browser`： 浏览器自动化工具，用于网页交互、表单填写、截图等任务
- `diagram-generator`： 生成 Draw.io 架构图（总体、应用、数据、技术、部署、集成图）
- `docx`： Word 文档创建、编辑和分析，支持修订、批注和格式处理
- `excalidraw-diagram-generator`： 生成 Excalidraw 格式图表（流程图、关系图、思维导图等）
- `pdf`： PDF 处理工具包，支持文本提取、表单填写、合并拆分等
- `pocketstack`： PocketStack 全栈开发框架，基于 React + shadcn/ui + PocketBase
- `pptx`： PowerPoint 演示文稿创建、编辑和分析
