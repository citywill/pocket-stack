# 用户管理

## 模块概述

用户管理模块，用于管理系统中的用户账号。包含用户数据概览和用户列表管理，面向管理员使用。

## 功能特性

- **数据概览**：展示活跃用户数、总用户数、用户注册趋势图表、最新注册用户列表
- **用户列表**：支持用户搜索、分页浏览、创建/编辑/删除用户、头像上传、密码重置、邮箱验证状态管理

## 页面

| 页面 | 路径 | 说明 |
|------|------|------|
| 数据概览 | `/user` | 用户统计数据与趋势图表 |
| 用户列表 | `/user/list` | 用户卡片列表，支持 CRUD 操作 |

## 权限

仅管理员（adminOnly）可访问。

## 数据模型

本模块直接使用 PocketBase 内置的 `users` collection，无需额外迁移。
