# 一个后台框架

当前项目用 shadcn create 创建项目，下面的命令定义了项目创建的参数，请遵守该风格协定，不要轻易修改。

```bash
npx shadcn@latest create --preset "https://ui.shadcn.com/init?base=radix&style=maia&baseColor=neutral&theme=blue&iconLibrary=lucide&font=inter&menuAccent=subtle&menuColor=default&radius=large&template=next" --template next
```

## 风格

- 使用 shadcn/ui
- 使用 blue 主题色
- 使用 maia 风格
- 使用 default 圆角

## 后端

使用 pocketbase 实现后端认证和接口服务。
