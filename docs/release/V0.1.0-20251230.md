# V0.1.0（2025年12月版本）发布说明

PocketStack 项目正式启动，完成了从零到一的搭建。本月建立了项目基础设施，包括管理后台框架、用户系统、权限体系、任务管理、VitePress 文档站以及 Docker 部署方案，为后续模块化开发奠定了坚实基础。

## 新特性

### 系统框架

- 项目初始化：基于 React 19 + Vite 7 + TypeScript + Tailwind CSS v4 初始化 PocketStack 项目。
- PocketBase 集成：对接 PocketBase 后端，实现超级管理员登录认证。
- 管理后台框架：实现 Header、Sidebar、MainLayout 三大核心布局组件。
- 侧边栏折叠：支持侧边栏折叠/展开，提升内容区域利用率。
- 仪表盘：管理员仪表盘（用户统计）和普通用户仪表盘（任务统计图表、快速操作）。
- MIT License：添加开源许可。
- 404 页面：添加 404 页面及路由处理。
- 自定义 favicon：添加项目 favicon。
- Logo：添加项目 Logo 和二级菜单支持。

### 用户管理及认证

- 用户管理：超级管理员可管理所有用户，支持分页。
- 权限系统：区分超级管理员和普通用户，支持登录切换。
- 用户注册：支持新用户自主注册。
- 个人中心：普通用户可查看和修改个人信息、头像显示、修改密码。

### VitePress 文档站

搭建项目文档站点，支持 GitHub Pages 部署。

### 主题切换

实现黑夜模式/白天模式切换功能。

### Docker 部署

添加 Docker 和 Nginx 配置文件用于生产部署。

### 任务管理模块

完整的任务 CRUD 功能，包含列表、看板视图、日历视图。
