# 2026年4月版本发布说明

## 新特性

### 模块管理

新增模块管理功能，支持模块的初始化、发现和系统状态管理。

### 用户管理模块

独立用户管理模块，包含路由和菜单配置。

### 系统设置模块

重构系统设置模块为独立模块，迁移全局配置功能。

### 模块包定义

新增模块 package.json 元数据文件，标准化模块依赖声明。

### 图标选择器

新增 HeroIcons 图标选择器组件。

### UI 组件扩展

新增进度条（Progress）、手风琴（Accordion）、工具提示（Tooltip）、树选择（TreeSelect）组件。

## 功能增强

- 重构文档结构并合并开发教程内容
- 完善示例模块文档，补充详细内容
- 更新模块文档和导航结构
- 重构模块说明文档并调整导航配置

## 缺陷修复

- 修复模块初始化和显示时使用错误的名称字段
- 清理未使用的导入和优化代码结构
