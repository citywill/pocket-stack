# 2026年4月版本发布说明

本月围绕模块化和组件化两个方向推进。新增了模块管理功能，将用户管理和系统设置拆分为独立模块，标准化了模块包定义。同时扩展了 UI 组件库，新增图标选择器、进度条、手风琴等多个实用组件。

## 新特性

### 模块管理

新增模块管理功能，支持模块的初始化、发现和系统状态管理。

![模块列表](../modules/screenshots/modules-list.png)

### 模块新增包定义文件

新增模块 package.json 元数据文件，标准化模块依赖声明。

### 系统功能模块化

- 用户管理模块：独立用户管理模块，包含路由和菜单配置。
- 系统设置模块：重构系统设置模块为独立模块，迁移全局配置功能。

### 系统组件

- 图标选择器：新增 HeroIcons 图标选择器组件。
- UI 组件扩展：新增进度条（Progress）、手风琴（Accordion）、工具提示（Tooltip）、树选择（TreeSelect）组件。
