# V0.3.0 发布说明

*2026年2月24日发布*

---

本月聚焦于模块系统的自动化和示例丰富化。实现了路由和菜单的自动注册机制，新增了包括 AI 对话、表单组件在内的多个示例页面，同时增强了菜单的显示控制和外部链接能力。文档方面完成了结构重构，补充了大量演示截图。

## 新特性

### 自动路由与菜单注册

新增模块后路由和菜单自动注册，无需手动修改主入口。

### 菜单增强

支持菜单显示/隐藏控制、外部链接跳转、自定义激活路径匹配规则。

### 示例模块

- 示例模块：统一示例模块管理，包含 AI 对话、表单组件、加载页面等。
- AI 对话示例：支持 Markdown 渲染的聊天对话界面。
- 表单组件：新增丰富的表单组件示例，包含输入框、单选、多选、下拉、日期、滑块、开关等。
- 整体加载示例：新增 Loading 加载状态示例页面。

## 功能增强

- 系统：重构侧边栏菜单配置，提取到独立文件
- 文档
  - 拆分模块开发文档为独立文件
  - 文档结构重构，添加各类演示截图（口袋看板、笔记、记账、OKRs、AI 助理等）
  - 为文档图片添加缩放功能

## 缺陷修复

- 路由：修复保护路由组件支持可选 children 属性
- 设置：禁用系统设置更新请求的自动去重
- 示例：改进 AI Chat 滚动逻辑
