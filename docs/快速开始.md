# 快速开始

## 使用skill开始

使用`pocketstack skill`，即可创建一个PocketStack项目。

例如，在agent中输入提示词：`在{指定目录下}初始化一个PocketStack项目，并运行开发环境`

## 手动开始

### 下载代码

```bash
git clone https://github.com/citywill/pocket-stack.git
cd pocket-stack
```

### 启动后端 (PocketBase)

**下载 PocketBase 二进制文件**

Pocketbase 下载地址：https://pocketbase.io/docs/

建议将pocketbase文件存放在项目 `.pocketbase/` 目录下。

### 启动前端

**1. 配置环境变量**

```bash
cp .env.example .env
```
创建 `.env` 文件，根据需要修改其中的变量。

其中 `VITE_POCKETBASE_URL` 为 PocketBase 后端地址。

**2. 安装依赖**

```bash
npm install
```

**3. 启动开发服务器，同时启动 PocketBase 后端**

```bash
npm run dev
```

默认网址为： `http://localhost:5173`
